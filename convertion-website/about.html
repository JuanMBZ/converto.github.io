ABOUT This Page

Converto, as you may see from the design and elements of this webpage, is a beginner-level site, written using HTML and CSS for the design and mechanics of the site, and Javascript for the calculations used for the conversions.

There is a huge lack in the number of conversions the site can offer, and there were a lot of planned features that didn't make it due to my lack of skills in Javascript. But for a first web project I would say I'm satisfied enough with this outcome, and I've also learned a lot along the way.
I'll try to do better next time.
